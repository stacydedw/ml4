# Анализ и классификация датасета Iris

## Описание
Набор данных Iris использовался в классической статье Р. А. Фишера 1936 года « Использование множественных измерений в таксономических задачах», а также доступен в репозитории машинного обучения Калифорнийского университета в Ирвайне (UCI Machine Learning Repository).  
  
Включает три вида ирисов, по 50 образцов каждого, а также некоторые характеристики каждого цветка. Один вид цветков линейно отличается от двух других, но два других вида нелинейно отличаются друг от друга.  

## Основные возможности
- EDA: сводная статистика, проверка пропусков, анализ распределений
- Предобработка: кодирование целевой переменной, стандартизация (StandardScaler) и нормализация (MinMaxScaler)
- Визуализация: гистограммы исходных/масштабированных данных, тепловые карты корреляций
- Корреляционный анализ: матрица Пирсона для всего датасета и в разрезе каждого класса
- Снижение размерности: PCA (сохранение 95.81% дисперсии в 2 компонентах) и MDS
- Классификация: логистическая регрессия
